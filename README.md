## Usage for Colorize

```bash
fmt.Println(Colorize("Success", "success"))
fmt.Println(Colorize("Failed", "fail"))
fmt.Println(Colorize("Warning", "warn"))
fmt.Println(Colorize("Note", "note"))
fmt.Println(Colorize("Default", ""))
```
